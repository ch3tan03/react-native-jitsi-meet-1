# ko-react-native-jitsi-meet

Jitsi for KO

## Installation

```sh
npm install ko-react-native-jitsi-meet
```

## Usage

```js
import KoReactNativeJitsiMeet from "ko-react-native-jitsi-meet";

// ...

const result = await KoReactNativeJitsiMeet.multiply(3, 7);
```
